— category: clinical | psychotherapy | nonclinical
— supervision_type: individual | group | none
— supervisor_name: must match a supervisor name in Settings

GitHub Sync

Your entries and settings are saved as JSON files in your private GitHub repository. Data syncs automatically on every save and loads fresh when you open the app.

Needs repo scope. Token is stored only in this browser, never synced.
How to set this up ›
Step 1 — Create a private repo
Go to github.com/new, name it anything (e.g. lcsw-tracker-data), set it to Private, check "Add a README", click Create.

Step 2 — Generate a Personal Access Token
Go to github.com/settings/tokens/new
• Note: LCSW Tracker
• Expiration: No expiration (or 1 year)
• Scopes: check repo (full control of private repositories)
• Click "Generate token" — copy it immediately, you won't see it again.

Step 3 — Connect
Enter your GitHub username, repository name, and the token above, then click Save & Connect. The app will create data/entries.json and data/settings.json in your repo automatically.

⚠ Token security
Your token is stored only in this browser's localStorage and is never written to GitHub. Don't share your token or use this app on a public/shared computer without locking it first.
