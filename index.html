Import from CSV

Your CSV must have these columns in this order:

date, hours, category, supervision_type, supervisor_name, employer, notes

— date: YYYY-MM-DD format (e.g. 2024-03-15)
— hours: number, increments of 0.5 (e.g. 3.5)
— category: clinical | psychotherapy | nonclinical
— supervision_type: individual | group | none
— supervisor_name: must match a supervisor name in Settings
— employer: text
— notes: text (can be blank)

Your entries and settings are saved as JSON files in your private GitHub repository.

Needs repo scope. Token stored only in this browser.
How to set this up ›
Step 1 — Create a private repo at github.com/new.

Step 2 — Generate a token at github.com/settings/tokens/new with repo scope.

Step 3 — Enter your username, repo name, and token above, then click Save & Connect.
